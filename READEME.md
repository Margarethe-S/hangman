# 🎮 Hangman – Web Game Project

![tested](https://img.shields.io/badge/tested-local-blue)
![status](https://img.shields.io/badge/status-learning_project-orange)

**Tech Stack:** HTML · CSS · JavaScript · Node.js

## 🧠 Project Overview

Hangman is a small web-based learning project created during training.

The goal was to practice:

- JavaScript logic & game mechanics
- DOM manipulation
- basic Node.js server setup
- UI structure with HTML & CSS
- handling user input and game state

Players can guess letters, see remaining attempts, and even add their own custom word.

---

## 🎮 Features

- Random hidden words
- Letter guessing system
- Visual hangman drawing
- Remaining attempts counter
- List of guessed letters
- Custom word input
- Restart / new game function
